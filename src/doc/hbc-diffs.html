HBC - differences and extensions

HBC 0.9999.4 is a Haskell 1.4 implementation with a few extensions. These extensions are normally turned off, but may be turned on with compiler flags. There are also some minor differences with Haskell 1.4. All of this is described here; the structure of the description follows that of the report.
